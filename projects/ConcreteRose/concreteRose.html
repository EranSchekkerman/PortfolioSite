Concrete Rose

About Game

Concrete Rose is a Hip-Hop jRPG game currently in development by Coquito Games. During my graduation assignment I developed a dynamic social system and created a complementary narrative demo using Unity and Articy:Draft X.

Social Link System

Concrete Rose is a game with a confidant system much like the Persona series, though there is a few large differences I am implemented during my work at Coquito Games.

I created the biggest feature of this system with a focus on player choices and consequences. The player chooses to add NPCs to their team and could very well choose to fight alone and unlock a hidden ending. Player choice is also encouraged by the relationship-based level system that lets players choose in-game buffs they can use in battles.

Furthermore I designed a dynamic social variable system that gave NPC extra character traits by the use of love language modifiers, which act as a double edged sword when it comes to social interactions with the NPCs.

Dialogue Implementation

For the project I wrote 10 days of dummy dialogue for 2 seperate characters, including player choices and consequences. I wrote inside Articy:Draft X and implemented into Unity using the Articy:Draft importer for Unity, and matched it to the project needs.

UI Creation

I designed and implemented the UI for the narrative demo and matched it to the style of the game to further convey to themes of the game. I then made it controller friendly to suit the main target console.
